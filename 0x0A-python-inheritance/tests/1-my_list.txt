1-my_list
=====================

Imports
    >>> MyList = __import__('1-my_list').MyList

Prints a sorted list
    >>> my_list = MyList()
    >>> my_list += [0]
    >>> my_list.append(5)
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [0, 1, 2, 3, 4, 5]

Empty list 
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Returns a new list
    >>> my_list = MyList()
    >>> my_list.append (9)
    >>> my_list.append (1)
    >>> my_list.append (8)
    >>> my_list.print_sorted()
    [1, 8, 9]
    >>> my_list
    [9, 1, 8]
